# ExplosionFilter

## プラグイン説明
指定したブロックを爆発による破壊から保護します

## コマンド
| コマンド名   |     説明      | 権限 |
| --- | ----------- | ------- |
| /explosionfilter open | 設定用GUiを開く | explosionFilter.admin |

## 使い方
/explosionfilter open

を実行すると設定GUIが開きます。

GUI内に爆発から保護したいブロックを配置して閉じると設定が保存されます。

 
## 開発環境
- Minecraft Version : 1.20.1
- Kotlin Version : 1.8.0
